# Immo_Webscraping
Schreibt einen auf Pythons Selenium basierenden Bot, um Daten zu zum Verkauf stehenden Häusern von ImmobilienScout24 zu sammeln. Es sei darauf hingewiesen, dass der Code nur zur Veranschaulichung dient und nicht aktiv genutzt werden sollte, da der Webseiten-Betreiber Richtlinien gegen Webscraping haben könnte.
